# 灵犀 (Lingxi) - 智慧调度系统

> **"心有灵犀，一点就通"** ✨

## 🎯 核心理念

**一句话，全自动** - 用户只需说一句话，系统自动：
1. 理解意图
2. 拆解任务
3. 分配角色
4. 并行执行
5. 汇总反馈
6. 评分学习

## 🔧 使用方式

### Python 调用

```python

# 初始化
orch = SmartOrchestrator()

# 执行任务
result = await orch.execute("帮我写个小红书文案，配张性感自拍")
```

## ⚙️ 评分系统

每个子任务执行完成后，丝佳丽会对其进行评分（满分10分）：

| 评分项 | 分值 | 说明 |
|--------|------|------|
| 输出完整性 | +1.0 | 输出数据完整 |
| 执行速度 | +1.0 | 响应及时 |
| 结果质量 | +1.0 | 质量良好 |
| 基础分 | 7.0 | 默认基础分 |

**评分示例**:
- 9.0-10.0: 优秀，超出预期
- 8.0-8.9: 良好，符合预期
- 7.0-7.9: 合格，基本满足
- <7.0: 待改进，需要优化

## 💡 优化建议

1. **意图识别增强** - 使用大模型进行深度意图理解
2. **动态角色注册** - 支持运行时添加/删除角色
3. **成本优化** - 简单任务用便宜模型，复杂任务用好模型
4. **错误恢复** - 角色失败自动切换备用角色
5. **学习进化** - 根据评分反馈，角色自我优化
